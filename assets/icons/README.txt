Put your custom icon images in these folders:

- certifications/  -> icons for each certification item
- education/       -> icons for each education item
- experience/      -> icons for experience entries

Recommended format:
- SVG (best) or PNG
- Square icons (e.g., 64x64, 128x128)
- Transparent background preferred

Example filenames:
- certifications/ibm-ai.svg
- certifications/cybersecurity.png
- education/nhce-logo.svg
- education/kle-logo.png
- experience/toyota.svg
